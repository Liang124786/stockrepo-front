# CatStocks – Frontend

台股資料視覺化與查詢系統的前端專案，  
提供產業分布、個股查詢、自選股與使用者介面。

## Tech Stack
- Vue 3
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- Element Plus
- Apache ECharts

## Features
- 產業樹狀圖（Treemap）
- 個股搜尋 / 個股頁
- 自選股管理
- 登入 / 註冊 / 個人檔案
- RWD（桌機 / 手機，手機使用漢堡選單）

## Deployment
- GitHub Pages
- CI/CD：GitHub Actions
